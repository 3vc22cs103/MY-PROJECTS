# Placement Readiness Platform

A web application designed to help students prepare for campus placements by providing resources for coding practice, aptitude preparation, and interview readiness.

## 🚀 Features

- Placement preparation dashboard
- Coding practice resources
- Aptitude preparation
- Interview preparation materials
- Simple and responsive user interface

## 🛠 Technologies Used

- React.js
- Vite
- HTML
- CSS
- JavaScript
- Node.js
- Git & GitHub
